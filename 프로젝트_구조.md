# 유튜브 쇼츠 자동 편집기

## 개요
이미지 + TTS + 대본 + 틀(프레임)을 조합하여 유튜브 쇼츠 영상을 자동 생성하고, 8개 채널에 동시 업로드하는 웹 기반 도구.

- **서버:** Flask (포트 8000, 내부 IP 접속 가능)
- **접속:** `http://192.168.0.7:8000`

---

## 설치 방법

```bash
# 1. 저장소 클론
git clone https://github.com/kwings90-hub/youtube-shorts.git
cd youtube-shorts

# 2. Python 패키지 설치
pip install -r requirements.txt

# 3. ffmpeg 설치 (시스템 패키지)
# macOS
brew install ffmpeg
# Ubuntu/Debian
sudo apt install ffmpeg
# Windows
winget install ffmpeg

# 4. 인증 파일 추가 (직접 복사)
# - client_secret.json (Google OAuth 계정A)
# - client_secret_B.json (Google OAuth 계정B)
# - youtube_auth/ 폴더 (채널별 토큰)

# 5. 서버 실행
python3 app.py
```

## 모듈별 상세

### app.py — 웹 서버
- Flask 서버 + 전체 UI (HTML/CSS/JS 인라인)
- 세션 관리로 업로드 파일 격리
- SSE(Server-Sent Events)로 생성 진행률 실시간 전송
- 최대 업로드: 500MB

**API 엔드포인트:**

| 메서드 | 경로 | 기능 |
|--------|------|------|
| GET | `/` | 메인 UI |
| GET | `/api/check` | 세션 상태 확인 |
| POST | `/api/upload/script` | 대본 업로드/저장 |
| POST | `/api/upload/images` | 이미지 업로드 |
| POST | `/api/delete/image` | 이미지 삭제 |
| POST | `/api/clear/images` | 이미지 전체 삭제 |
| POST | `/api/upload/tts` | TTS 오디오 업로드 |
| POST | `/api/session/new` | 새 세션 생성 |
| GET | `/api/session/info` | 세션 정보 조회 |
| POST | `/api/generate` | 영상 생성 시작 |
| GET | `/api/progress` | SSE 진행률 스트림 |
| GET | `/output/<filename>` | 생성된 영상 다운로드 |
| GET | `/api/videos` | 생성된 영상 목록 |
| GET | `/api/youtube/channels` | 인증된 채널 목록 |
| POST | `/api/youtube/upload` | YouTube 업로드 |

### auto_editor.py — 영상 생성 엔진
- 영상 규격: 1080x1920 (세로), 24fps
- Whisper 기반 자막 싱크
- CLIP 기반 자동 이미지-텍스트 매칭 (한국어)
- OpenCV 얼굴 감지 크롭
- 채널별 틀(프레임) + 자막/헤드라인 색상 적용
- 폰트: AppleSDGothicNeo (Bold/SemiBold)

### youtube_uploader.py — 업로드
- 10MB 청크 단위 Resumable Upload
- 진행률 콜백 지원 (0~100%)
- 자동으로 `#Shorts` 태그 추가
- 예약 공개(publishAt) 지원
- 에러 처리: 할당량 초과, 권한 없음

### youtube_auth.py — 인증
- 채널별 개별 OAuth 토큰 발급/저장
- 계정A/B 자동 분기
- 토큰 자동 갱신

### delogo_marker.py — 로고 제거 GUI
- tkinter 기반 GUI 도구
- 사각형/타원/프리핸드 마킹
- ffmpeg delogo 필터용 좌표 JSON 출력

## 영상 생성 워크플로우

```
1. 웹 UI에서 이미지/TTS/대본 업로드
2. 채널 선택 → "생성" 버튼
3. auto_editor.py 실행:
   - Whisper로 TTS 음성 → 자막 타임스탬프
   - CLIP으로 이미지-대본 자동 매칭
   - 채널별 틀(프레임) 합성
   - OpenCV 얼굴 감지 → 이미지 크롭
   - ffmpeg로 최종 영상 인코딩
4. SSE로 진행률 실시간 표시
5. 생성 완료 → YouTube 업로드 (선택)
```
